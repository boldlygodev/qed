# qed Examples

Practical examples of `qed` in common use cases.
Examples use named patterns where they aid readability and inline patterns for one-liners.

---

## Code Editing

**Remove all TODO comments:**

```sh
qed 'at(/^\s*\/\/\s*TODO:/) | qed:delete()' main.go
```

**Uppercase all constants:**

```sh
qed 'at(/^const /) | qed:upper()' constants.go
```

**Add a license header to every Go file:**

```sh
#!/usr/bin/env qed -f
before() | cat /path/to/license-header.txt
```

**Delete a function by name:**

```sh
qed --in-place '
func_start=/^func handleRequest\(/
func_end=/^\}/
from(func_start+) > to(func_end+) | qed:delete()
' main.go
```

**Add a deprecation notice above every function matching a pattern:**

```sh
qed --in-place '
target=/^func Old/
before(target) | echo "// Deprecated: use New* equivalent instead."
' main.go
```

**Indent a selected region:**

```sh
qed 'from(/^func /+) > to(/^\}/+) | qed:indent(4)' main.go
```

**Remove trailing whitespace:**

```sh
qed 'at(/\s+$/) | qed:replace(/\s+$/, "")' file.txt
```

## AI-Assisted Transformation

`qed` composes naturally with AI CLI tools like
[`llm`](https://github.com/simonw/llm) via the processor pipeline.
The selected region is passed as stdin — the AI's output replaces it.

**Implement a TODO:**

```sh
qed --in-place '
todo=/\/\/ TODO:.*/
at(todo) | llm "implement this based on the comment"
' main.go
```

**Add error handling to a function:**

```sh
qed --in-place '
func_start=/^func /
func_end=/^\}/
from(func_start+) > to(func_end+) |
    llm "add idiomatic Go error handling to this function"
' main.go
```

**Rewrite comments to be more concise:**

```sh
qed --in-place 'at(/^\/\/.*/) | llm "rewrite this comment more concisely"' main.go
```

**Generate a docstring for a function:**

```sh
qed --in-place '
func_start=/^func /
before(func_start) |
    llm -s "You are a Go expert. Write a godoc comment for this function signature." \
        --stdin-as-file
' main.go
```

**Translate comments to another language:**

```sh
qed --in-place 'at(/^\/\/.*/) | llm "translate this comment to French"' main.go
```

## Config File Manipulation

**Update a value in a TOML file:**

```sh
qed --in-place '
at(/^version = /) | qed:replace(/=.*/, "= \"2.0.0\"")
' Cargo.toml
```

**Comment out a section in an INI file:**

```sh
qed --in-place '
section=/^\[database\]/
section_end=/^\[/
from(section+) > to(section_end) | qed:prefix(text:"# ")
' config.ini
```

**Replace a value in a YAML file:**

```sh
qed --in-place '
at(/^  image:/) | qed:replace(/:.*/, ": myrepo/myimage:${IMAGE_TAG}")
' deployment.yaml
```

**Delete a YAML key block:**

```sh
qed --in-place '
key=/^  annotations:/
next_key=/^  [a-z]/
from(key+) > to(next_key) | qed:delete()
' deployment.yaml
```

## Log Processing

**Extract only error lines:**

```sh
qed --extract 'at(/\bERROR\b/)' app.log
```

**Delete debug lines:**

```sh
qed 'at(/\bDEBUG\b/) | qed:delete()' app.log
```

**Add line numbers to a log:**

```sh
qed 'at() | qed:number()' app.log
```

**Extract a time range from a log:**

```sh
qed --extract '
start=/2026-02-26 09:00/
end=/2026-02-26 10:00/
from(start+) > to(end+) | qed:skip()
' app.log
```

**Reformat a timestamp in log lines:**

```sh
qed 'at(/^\d{4}-\d{2}-\d{2}/) | qed:replace(/(\d{4})-(\d{2})-(\d{2})/, /$3\/$2\/$1/)' app.log
```

## Code Generation

**Inject generated code into a Go file:**

```go
//go:generate qed --in-place -f generate-routes.qed routes.go
```

```sh
#!/usr/bin/env qed -f
# generate-routes.qed
# Replaces the region between generation markers with fresh output

marker_start=/\/\/ CODE GENERATED START/
marker_end=/\/\/ CODE GENERATED END/

from(marker_start+) > to(marker_end) |
    ./scripts/generate-routes.sh
```

**Insert a generated file header:**

```sh
qed --in-place '
before() | printf "// Code generated by qed. DO NOT EDIT.\n// Source: %s\n\n" "${SOURCE_FILE}"
' generated.go
```

**Stamp a build version into source:**

```sh
qed --in-place '
at(/^var Version = /) | qed:replace(/=.*/, "= \"${BUILD_VERSION}\"")
' version.go
```

## Template Rendering

**Replace placeholders with environment variables:**

```sh
qed '
at(/\{\{APP_NAME\}\}/) | qed:replace("{{APP_NAME}}", "${APP_NAME}")
at(/\{\{VERSION\}\}/)  | qed:replace("{{VERSION}}", "${VERSION}")
' template.yaml
```

**Inject a UUID into a template:**

```sh
qed 'at(/\{\{uuid\}\}/) | qed:replace("{{uuid}}", qed:uuid())' template.sql
```

**Inject a timestamp:**

```sh
qed '
at(/\{\{generated_at\}\}/) |
    qed:replace("{{generated_at}}", qed:timestamp(format:datetime, timezone:UTC))
' report-template.md
```

**Inject a random token:**

```sh
qed 'at(/\{\{token\}\}/) | qed:replace("{{token}}", qed:random(32, alphabet:base62))' config-template.yaml
```

## Document Processing

**Remove all HTML comments from a markdown file:**

```sh
qed 'at(/<!--.*-->/) | qed:replace(/<!--.*-->/, "")' README.md
```

**Promote all headings by one level:**

```sh
qed 'at(/^##/) | qed:replace(/^##/, "#")' doc.md
```

**Extract all code blocks:**

```sh
qed --extract '
fence=/^```/
from(fence+) > to(fence+)
' doc.md
```

**Wrap long lines:**

```sh
qed 'at(/^[^|].{80,}/) | qed:wrap(width:80)' README.md
```

**Add a section to a markdown file:**

```sh
qed --in-place '
after(/^## Installation/) | printf "\n_Last updated: %s_\n" "$(date +%Y-%m-%d)"
' README.md
```

## $EDITOR Integration

`qed` can be used as `$EDITOR` for programs that invoke a non-interactive editor
to transform a file — `git`, `cron`, `kubectl`, `mutt`, and others.

**Git commit message:**

```sh
# ~/.config/qed/git-commit.qed
#!/usr/bin/env qed -f

# Trim trailing whitespace from every line
at(/\s+$/) | qed:replace(/\s+$/, "")

# Remove comment lines git adds
at(/^#/) | qed:delete()
```

```sh
export EDITOR="qed --in-place -f ~/.config/qed/git-commit.qed"
```

**kubectl edit — enforce resource limits:**

```sh
#!/usr/bin/env qed -f
# enforce-limits.qed
# Ensures resource limits are set on every container spec

limits=/^\s*limits:/
after(limits) |
    printf "          cpu: \"500m\"\n          memory: \"256Mi\"\n"
```

```sh
EDITOR="qed --in-place -f enforce-limits.qed" kubectl edit deployment myapp
```

**Crontab — add logging to every job:**

```sh
qed --in-place '
job=/^[^#].*/
at(job) | qed:replace(/$$/, " >> /var/log/cron.log 2>&1")
' /tmp/crontab.tmp
```

**mutt — sign outgoing emails:**

```sh
#!/usr/bin/env qed -f
# mutt-sign.qed
after() | cat ~/.signature
```

```sh
export EDITOR="qed --in-place -f ~/.config/qed/mutt-sign.qed"
```
